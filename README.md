# dnlzrgz.com

A simple and fast OS-like portfolio made with Django, Wagtail, some pure CSS and a bit of JavaScript with almost no build process.

![screenshot of the home page](./.github/screenshot.png)

</div>

## Features

- 📜 Statically generated thanks to [`wagtail-bakery`](https://github.com/wagtail-nest/wagtail-bakery) and [`django-bakery`](https://github.com/palewire/django-bakery).
- 💡 Dark/light themes.
- ⚡ Responsive styles.
- 🪶 Ready for blogging.
- 📤 RSS feed.
- 🌱 SEO-friendly URLs, meta tags, LD+JSON and sitemap generation.
- 📦 Easy to deploy on AWS or platforms such as Cloudflare pages.
- ⚙️ Configurable via env variables.
